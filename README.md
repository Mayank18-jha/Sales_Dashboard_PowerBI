# Sales_Dashboard_PowerBI

An interactive **Power BI Dashboard** and **Python visualizations** built to analyze business data and present key insights through effective data storytelling.

## 🚀 Tools Used
- Power BI
- Python
- Pandas
- Matplotlib
- Seaborn

## 📌 Features
- Interactive Power BI Dashboard
- KPI Cards
- Sales Trend Analysis
- Category & Product Analysis
- Customer Insights
- Python Charts (EDA)
- Dashboard Screenshots (PNG)

## 📂 Files Included
- `Dashboard.pbix` – Power BI Dashboard
- `Python Charts/` – Visualization images
- `Dashboard.png` – Dashboard Preview
- `README.md`

## 🎯 Objective
Transform raw data into meaningful insights using interactive dashboards and Python-based visualizations.
